# Quotient

**한국·미국 자산을 한 화면에. 자연어로 묻고, 즉시 분석을 받으세요.**

블룸버그 터미널의 정보 밀도·미감을 개인 가격대로.

---

## 이런 분께

여러 증권사·앱에 흩어진 한국·미국 주식을 한 곳에서 보고 싶은 분.
정확한 수치와 빠른 응답을 원하는 분.
"오늘 내 포트폴리오 어때?"를 채팅하듯 물어보고 싶은 분.

## 무엇을 할 수 있나요

- **📊 통합 포트폴리오** — KOSPI·KOSDAQ·NASDAQ·NYSE 자산을 한 화면에. 원화 환산·손익·비중을 자동으로
- **💬 AI 분석가** — "내 포트폴리오의 통화 분산이 어때?", "삼성전자 최근 한 달 추이는?" 같은 질문에 실데이터 기반 답변
- **📈 마켓 모니터** — KOSPI·S&P 500·환율·금리 등 주요 지표를 한 눈에 (시세 15분 지연)
- **🌅 매일 아침 브리핑** — 보유 자산·관심 종목·시장 변화를 짧게 요약해 매일 07:00에 제공

## 약속

이 서비스는 분석과 정보 제공만 합니다.

- 💰 자금을 보관하거나 이체하지 않습니다
- 🔐 마이데이터로 계좌를 연동하지 않습니다 (보유 자산은 직접 입력)
- 📢 직접적인 매수/매도 추천을 하지 않습니다 — 분석 관점만 제공
- 🧑‍⚖️ 세금·법무 자문은 다루지 않습니다

## 가격

MVP 단계는 **무료**입니다. 가입자 100명 또는 일평균 사용 500회를 넘으면 유료 플랜(₩14,900/월)을 검토할 예정입니다. 광고는 기본 비활성.

## 면책

투자 결정은 본인의 책임입니다. 본 서비스는 정보 제공 목적이며 투자 자문업·금융상품판매업이 아닙니다.
시세 데이터는 공개 API 기반이며 15분 이상 지연될 수 있습니다.
